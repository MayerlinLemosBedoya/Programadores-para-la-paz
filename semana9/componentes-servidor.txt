Sistema operativo:
Es el software principal del computador o servidor. Se encarga de administrar los recursos del sistema y permitir que los programas funcionen correctamente.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Gracias a Node.js es posible crear y ejecutar servidores web.

Aplicación server.js:
Es el archivo donde se encuentra el código del servidor. Define las rutas, procesa solicitudes y envía respuestas a los usuarios.

Puerto:
Es un punto de comunicación que utiliza el servidor para recibir solicitudes. Por ejemplo, el puerto 3000 permite acceder a la aplicación mediante una dirección como localhost:3000.

Logs:
Son registros que almacenan información sobre la actividad y los eventos del servidor. Ayudan a detectar errores, monitorear el funcionamiento y solucionar problemas.
